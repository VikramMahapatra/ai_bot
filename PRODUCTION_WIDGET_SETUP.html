Embed this widget on any website to add an AI chat assistant

Quick Start

Add this code to your website before the closing </body> tag:

<script> window.AIChatbot = { widgetId: 'widget_1769582231293', apiUrl: 'https://your-api-domain.com', name: 'AI Assistant', welcomeMessage: 'Hi! How can I help you today?', primaryColor: '#007bff', position: 'bottom-right' }; </script> <script src="https://cdn.example.com/chatbot-widget.iife.js"></script>

Configuration Options

Option Type Description
widgetId string Unique identifier for this widget
apiUrl string Your backend API URL (e.g., https://api.yourdomain.com)
name string Display name in chat header
welcomeMessage string First message shown when user opens chat
primaryColor hex color Widget theme color (e.g., #007bff)
position string bottom-right or bottom-left

Deployment Steps

  1. Build the widget
    cd widget npm run build
  2. Upload to CDN or web server
    Upload: widget/dist/chatbot-widget.iife.js
  3. Update API URL
    apiUrl: 'https://your-live-api-domain.com'
  4. Add script tag to your website

    Copy the Quick Start code above to your website.

  5. Test on your domain

    Open your website and look for the chat bubble in the bottom-right corner.

Features

  • AI-powered responses with knowledge base integration
  • Conversation history (stored locally in browser)
  • Lead capture (optional)
  • Customizable colors and messages
  • Mobile-friendly and responsive
  • No dependencies - single IIFE script

Popular Hosting Options

  • GoDaddy - Upload chatbot-widget.iife.js to File Manager under /public_html/
  • AWS S3 + CloudFront - Upload to S3 bucket, serve via CloudFront CDN
  • Cloudflare - Use Cloudflare Pages or Workers
  • Your own server - Copy to /var/www/cdn/ and serve via your web server
