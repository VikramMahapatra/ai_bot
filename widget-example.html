AI Chatbot Widget Integration Example

This page demonstrates how to integrate the AI Chatbot widget into your website.

Integration Code

<!-- Add this script before closing </body> tag -->
<script>
  (function() {
    window.AIChatbot = {
      widgetId: 'demo-widget',
      apiUrl: 'http://localhost:8000',
      name: 'AI Assistant',
      welcomeMessage: 'Hi! How can I help you today?',
      primaryColor: '#007bff',
      position: 'bottom-right'
    };
    var script = document.createElement('script');
    script.src = 'http://localhost:5174/chatbot-widget.iife.js';
    script.async = true;
    document.head.appendChild(script);
  })();
</script>

Configuration Options

  • widgetId: Unique identifier for your widget
  • apiUrl: Backend API URL
  • name: Display name of the chatbot
  • welcomeMessage: First message shown to users
  • primaryColor: Main color of the widget (hex code)
  • position: 'bottom-right' or 'bottom-left'

Try It Out

The chatbot widget should appear in the bottom-right corner of this page. Click the chat icon to start a conversation!
